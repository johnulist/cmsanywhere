== Description ==
This simple module allows you tho show a cms content in the hook you want.

Just go to module config, select any existing cms page and go to hooks to change it's position. This module can show the content for ALL hooks in prestashop. Useful for home page, checkout, carrier, user area...

Created By: Maztch
Email : maztch@gmail.com

== Installation ==

1. Upload the plugin to your /modules/ directory.

2. Activate the plugin through the modules menu in Prestashop.

3. Go to configure and select the content from existing cms you want to shop.

4. Go to manage the hook and select where you want the content.
